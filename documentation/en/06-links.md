# ðŸ”— ig.me Links and QR Codes

Encourage starting conversations with direct links and QR codes.

### 1. Generating ig.me Links

`ig.me` links redirect the user directly to a conversation with you on Instagram. You can include referral parameters to know where the user is coming from.

```php
use ScriptDevelop\InstagramApiManager\Facades\Instagram;

// Simple link
$link = $account->getIgMeLink();

// Link with referral parameter (campaign)
$campaignLink = $account->getIgMeLink('summer_2024');
```

### 2. QR Codes

Generate a QR code that points to your `ig.me` link:

```php
// Generate 500x500px QR with reference
$qrCode = Instagram::link()->generateIgMeQrCode($account, 'offline_store', 500);

// The result is an image URL generated by Meta
echo '<img src="' . $qrCode . '">';
```

### 3. Referral Statistics

If you use `ref` parameters, you can check how many people have entered through each one:

```php
$stats = Instagram::link()->getReferralStats($account->instagram_business_account_id, 'summer_2024');
```
